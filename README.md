# Projet de jeu sur la console Playdate

## Description du projet

Ce projet consiste à développer un jeu vidéo sur la console Playdate, dans le cadre d’un projet universitaire à l’UQAC (Université du Québec à Chicoutimi).
L’objectif est de mener le projet de A à Z, de la conception de l’idée jusqu’au développement final du jeu.

Le développement Playdate permet l’utilisation de :
- Lua (via le SDK Playdate)
- C (optionnel, pour des besoins plus bas niveau ou de performance)
Pour ce projet, le développement sera principalement réalisé en Lua, le langage recommandé pour débuter sur Playdate et plus accessible pour le prototypage rapide.

Ce projet à été réalisé dans le cadre du cours de projet de l'UQAC par **Basile MONOD** et **Edgar MARTIN**, sous la suppervision du professeur **Damien BRUN**.


## Objectifs pédagogiques

Les objectifs du projet sont les suivants :
- Découvrir le développement de jeu sur Playdate
- Apprendre à structurer un projet de jeu complet
- Travailler en équipe sur un projet
- Créer une documentation en français sur la création de notre jeu, et sur la création de jeux Playdate en général.
